Terminologi



Repository betyder:

Repository är ett projekts mapp. Den innehåller alla projektfiler och dokumentation som också visar varje fils revisionshistorik. Repository kan också delas med andra medarbetare.


Branch betyder:

En branch finns i ett repository. Till exempel kan en programmerare ha en branch från huvudbranchen. Programmeraren kan arbeta fritt med sin branch utan att påverka huvudbranchen och "live"-versionen. När programmeraren är klar med sin branch kan hen slå ihop/merge den tillbaka till huvudbranchen och sedan kan man se ändringarna.
